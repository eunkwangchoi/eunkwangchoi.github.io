1. 구동 방법
		
        인터넷 앱을 실행하면 자주 방문한 사이트에 Turing Machine 페이지가 등록되어 있습니다. 실행하면 바로 작품이 구동됩니다. 

	커서가 상단에 위치하면 주소 표시줄이 사라지지 않습니다. 커서를 아래쪽으로 이동하여 주소 표시줄을 사라지게 만듭니다. 시간이 경과하면 커서도 사라집니다.

        수동으로 입력해야 할 경우 아래의 주소를 입력합니다.
		
        https://eunkwangchoi.com/showcase/app
		
		
		2. 일반적인 문제 해결

        캐시 오류로 작품이 느리게 구동하거나 동작을 멈출 수 있습니다. 인터넷 앱의 설정에서 캐시를 비우고 재가동하면 해결됩니다. 설정 화면은 인터넷 앱 주소창 우측의 점 단추를 눌러 나타나는 메뉴에서 진입할 수 있습니다.
		
        메뉴 ▶ 설정 ▶ 인터넷 사용 기록 모두 삭제 (화면 맨 하단)
		
        캐시를 비우면 자주 방문한 사이트가 사라지므로 주소를 수동으로 입력하여 재가동해야 합니다.
		
		
		3. 일반적인 설정 사항

        기기는 인터넷 앱만 사용할 수 있습니다. 나머지 앱은 잠금 설정되어 있습니다. 부득이한 경우 비밀번호 0000으로 잠금을 해제합니다.
        
        기기는 터치 잠금 설정이 되어 있습니다. 리모컨으로 조작해야 합니다. 부득이 터치 설정을 켜야 할 경우 아래 메뉴에서 터치 잠금을 해제합니다.
		
        설정 ▶ 전체 설정 ▶ 일반 ▶ 시스템 ▶ 기타 설정 ▶ 터치 잠금
